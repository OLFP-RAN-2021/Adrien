# funny lists

Formater les listes ordonnées. 

- square (carrés)
- round (ronds)

- blue
- green
- red
- yellow

```html

    <!-- flex en ligne commence en haut à gauche-->
    <ol class="funny">
        <li class="square blue">

        </li>
        <li>
            ... 
        </li>

    </ol>
```
