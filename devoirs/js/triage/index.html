Sur le principe du tri à bulle. Mais comme j'autorise la bulle à reculer, la métaphore ne marche plus très bien. Donc je l'ai renommé tri à chariot...

Tableau généré aléatoirement : 

    Tableau trié : 
    


    Description logique : 

    
        POUR ( iteration = 0 ; iteration < COMPTER(tableau) ; iteration + 1 )

            suivant = iteration + 1
            
            SI ( tableau[iteration] > tableau[suivant] )
                
                transfert = tableau[suivant]
                tableau[suivant] = tableau[iteration]
                tableau[iteration] = transfert
            
            FIN_SI

            precedant = iteration - 1

            SI ( tableau[precedant] > tableau[iteration] )

                POUR ( inverse = 0 ; inverse > 0 ; inverse - 1 )

                    precedant = inverse - 1

                    SI ( tableau[precedant] > tableau[inverse] )

                        transfert = tableau[precedant]
                        tableau[precedant] = tableau[inverse]
                        tableau[inverse] = transfert

                        CONTINUER_POUR

                    FIN_SI
                    
                FIN_POUR

            FIN_SI

        FIN_POUR
